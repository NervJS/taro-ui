# 开发计划

## v1.2.0（已发布）

- [x] Button, Form, Input, Textarea 组件支持小程序对应的组件属性，例如 Button 的 open-type

- [x] AtIcon 支持传入自定义的 ICON FONT

- [x] 组件优化
