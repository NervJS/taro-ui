# 介绍

----

`AT-UI-Nerv` 是`AT-UI` 组件库基于 `NervJS`的移植版本 

## 特性

- 基于 `Nerv` 开发 UI 组件
- 使用 npm + webpack + babel 的工作流，支持 ES2015
- CSS 样式独立，即使使用不同的框架实现，也都能保持统一的 UI 风格（ 详见：[AT-UI Style](https://github.com/at-ui/at-ui-style) ）
- 提供友好的 API，可灵活的使用组件

## 浏览器支持

- 现代浏览器和 IE9 及以上
- [Electron](http://electron.atom.io/)
- [NW.js](http://nwjs.io)
